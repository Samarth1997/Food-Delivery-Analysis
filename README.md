🍔 Food Delivery Analysis – Power BI Dashboard
A comprehensive Power BI dashboard designed to analyze food delivery operations, customer behavior, restaurant performance, and delivery efficiency.
This project provides actionable insights to improve service quality, optimize delivery times, and enhance customer satisfaction.

📊 Overview
The Food Delivery Analysis Dashboard consolidates data from multiple sources to visualize key metrics such as revenue, orders, delivery performance, and customer trends.
It helps stakeholders monitor business performance and make data-driven decisions.
📈 Key Metrics
Total Revenue: 165M

Total Orders: 200K

Average Order Value: 824.57

Average Delivery Time: 45 mins

Total Customers: 20K

Repeat Customers: 10K

Total Restaurants: 6K

Average Rating: 4.14
🧠 Insights
Delivery Efficiency: 51.88% of orders delivered on time; 48.12% delayed.

Customer Retention: Equal distribution between new and repeat customers.

Restaurant Performance: Dine-out services lead with 40% of total orders.

Revenue Concentration: Top cities contribute over 15M in revenue.

Top Restaurant Example: Empire Restaurant – 222K revenue, 262 orders, avg delivery time 45 mins.

🛠️ Tools & Technologies
Power BI Desktop – Dashboard design and data modeling

Power Query – Data transformation and cleaning

DAX – Calculated measures and KPIs

Excel / CSV – Data source integration

Power BI Service – Publishing and sharing reports
